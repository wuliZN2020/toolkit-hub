# 📖 书摘排版模板示例

本文档展示了三种新增的公众号排版模板效果。

---

## 1. 文艺清新风

**特点**：使用文艺符号，有「今日书摘」标题，每条摘抄前有装饰符，结尾有温馨寄语

```
◆ 今日书摘

《百年孤独》
[加西亚·马尔克斯]

────────────

✦ 生命中真正重要的不是你遭遇了什么，而是你记住了哪些事，又是如何铭记的。

✦ 过去都是假的，回忆是一条没有归途的路，以往的一切春天都无法复原。

✦ 无论走到哪里，都应该记住，过去都是假的，回忆是一条没有归途的路。

────────────
📖 阅读让生活更美好
```

---

## 2. 数字序号风

**特点**：摘抄带序号便于引用，使用粗分隔线，标题紧凑，结尾有价值观表达

```
📚《三体》刘慈欣

━━━━━━━━━━━━━━

01.
弱小和无知不是生存的障碍，傲慢才是。

02.
给岁月以文明，而不是给文明以岁月。

03.
你的无畏来源于无知。

04.
在中国，任何超脱飞扬的思想都会砰然坠地的，现实的引力太沉重了。

━━━━━━━━━━━━━━
💭 用阅读对抗虚无
```

---

## 3. 极简引用风

**特点**：极简设计无emoji，使用引号包裹摘抄，短分隔线，有操作提示文案

```
《活着》余华

——————

" 人是为了活着本身而活着，而不是为了活着之外的任何事物而活着。"

" 没有什么比时间更具说服力了，因为时间无需通知我们就可以改变一切。"

" 最初我们来到这个世界，是因为不得不来；最终我们离开这个世界，是因为不得不走。"

——————

长按复制 | 分享收藏
```

## 使用方法

### 1. 搜索书籍模式
1. 搜索书籍
2. 选择书籍并加载摘抄
3. 勾选喜欢的摘抄
4. 选择上述三种模板之一
5. 点击「复制选中」按钮

### 2. 纯文本排版模式
1. 切换到「纯文本排版」标签
2. 输入书名和作者（可选）
3. 粘贴你的摘抄文本（每行一条）
4. 点击「解析摘抄」
5. 勾选需要的摘抄
6. 选择排版模板
7. 点击「复制选中」按钮

## 技术实现

### 模板配置格式（JSON）

#### 文艺清新风
```json
{
  "header": "◆ 今日书摘\\n\\n",
  "titleFormat": "《{title}》\\n[{author}]\\n\\n",
  "divider": "────────────\\n\\n",
  "quotePrefix": "✦ ",
  "quoteSeparator": "\\n\\n",
  "footer": "\\n────────────\\n📖 阅读让生活更美好"
}
```

#### 数字序号风
```json
{
  "titleFormat": "📚《{title}》{author}\\n\\n",
  "divider": "━━━━━━━━━━━━━━\\n\\n",
  "quoteFormat": "{index}.\\n{content}",
  "quoteSeparator": "\\n\\n",
  "footer": "\\n━━━━━━━━━━━━━━\\n💭 用阅读对抗虚无"
}
```

#### 极简引用风
```json
{
  "titleFormat": "《{title}》{author}\\n\\n",
  "divider": "——————\\n\\n",
  "quoteFormat": "\\" {content} \\"",
  "quoteSeparator": "\\n\\n",
  "footer": "\\n——————\\n\\n长按复制 | 分享收藏"
}
```

### 配置字段说明

- `header`: 最顶部的标题（可选）
- `titleFormat`: 书名和作者的格式，支持 `{title}` 和 `{author}` 占位符
- `divider`: 分隔线
- `quotePrefix`: 每条摘抄前的前缀符号
- `quoteFormat`: 摘抄格式，支持 `{index}` 和 `{content}` 占位符
- `quoteSeparator`: 摘抄之间的分隔符
- `footer`: 结尾文案，支持 `{title}` 占位符

## 排版建议

### 适用场景

- **文艺清新风**：适合文学作品、散文、诗歌类书籍
- **数字序号风**：适合知识类、干货类书籍，便于读者引用
- **极简引用风**：适合哲理、名言类摘抄，强调内容本身

### 发布平台

所有格式都经过优化，适合：
- 微信公众号
- 微信朋友圈
- 小红书（如需小红书专属格式，请选择「小红书排版」模板）
- 个人博客

### 最佳实践

1. **摘抄数量**：建议每次选择 3-8 条摘抄
2. **文字长度**：单条摘抄 50-200 字为佳
3. **主题统一**：选择主题相关的摘抄，不要混杂
4. **书籍信息**：尽量填写完整的书名和作者
5. **定期发布**：建立阅读分享的习惯
